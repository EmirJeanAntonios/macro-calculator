**Project Description**

This project is for calculating macros based on user's input.
The user can enter

- Age
- Biological Gender
- Weight
- Height
- Goal
- Workout Schedule

It will return the correct macro values, Kcal that the user should consume. If the macros has special days it should return those also


**Tech Stack**

this project is using reactjs for frontend, nestjs for backend and postgresql for database.


**Features**
- the backend can return PDF file
- api should return macros, kcal, if there are any special occasion it should return that also
- the user can select their workout and specify what they are doing on front end. it should be like a calendar, the user also should select the workout type and they can enter the hours.
